# Byte Blaze: AI-Based Maternal Health Risk Assessment Chatbot

**Byte Blaze** एक वेब-आधारित चैटबोट प्रोजेक्ट है, जो **गर्भवती महिलाओं के स्वास्थ्य जोखिमों का आकलन** करने में मदद करता है। इस प्रोजेक्ट का उद्देश्य माताओं को सुरक्षित और समय पर स्वास्थ्य सुझाव देना है।

## Features

- **Doctor Appointment Page:** उपयोगकर्ता डॉक्टर से अपॉइंटमेंट बुक कर सकते हैं।  
- **Sakhi Main Page:** AI चैटबोट जो मातृ स्वास्थ्य के बारे में प्रश्नों का उत्तर देता है।  
- Responsive और user-friendly UI।  
- Simple HTML, CSS और JavaScript based implementation।
